STUDY 1: BAFFLE SPACING RATIO (0.3 - 0.8)
  ✓ 1_bafflespacingratio.png (for reports/presentations)
  ✓ 1_bafflespacingratio.fig (editable MATLAB figure)
  ✓ 1_bafflespacingratio.mat (raw data)

  Key Finding: CRITICAL PARAMETER
  - Increasing from 0.4 → 0.8 reduced ΔP_s from 2177 → 53 kPa
  - This was the main problem with original design!

STUDY 2: FIN PITCH (3 - 6 mm)
  ✓ 2_pf.png
  ✓ 2_pf.fig
  ✓ 2_pf.mat

  Key Finding: Optimal at 5mm
  - Too many fins (3mm) causes flow blockage
  - 5mm balances area density with flow resistance

STUDY 3: NUMBER OF TUBE PASSES (2, 4, 6, 8)
  ✓ 3_Np.png
  ✓ 3_Np.fig
  ✓ 3_Np.mat

  Key Finding: Optimal at 4 passes
  - Balances U coefficient vs tube-side pressure drop
  - More passes = better U but quadratically higher ΔP

STUDY 4: TUBE INNER DIAMETER (19 - 22 mm)
  ✓ 4_Dinternal.png
  ✓ 4_Dinternal.fig
  ✓ 4_Dinternal.mat

  Key Finding: All standard sizes work well
  - 22mm ID gives lowest ΔP_t with minimal U penalty
  - All are EU metric standard sizes

STUDY 5: TUBE LENGTH (1.0 - 2.0 m)
  ✓ 5_Ltube.png
  ✓ 5_Ltube.fig
  ✓ 5_Ltube.mat

  Key Finding: Shorter is better
  - 1.0m gives best U and lowest pressure drops
  - More compact overall design

STUDY 6: FIN HEIGHT (3 - 8 mm)
  ✓ 6_lf.png
  ✓ 6_lf.fig
  ✓ 6_lf.mat

  Key Finding: 5mm is good balance
  - Taller fins have lower efficiency
  - Diminishing returns above 5mm

STUDY 7: FIN THICKNESS (0.5 - 2.0 mm)
  ✓ 7_tf.png
  ✓ 7_tf.fig
  ✓ 7_tf.mat

  Key Finding: 1mm is standard
  - Thicker gives marginal improvement
  - 1mm is typical for aluminum fins
